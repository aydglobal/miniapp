# ADN Control Panel Pack

Bu paket, oyunun canlı yönetimi için admin control panel iskeleti içerir.

## Kapsam
- analytics summary
- live tuning config
- event control
- mission toggle
- economy override
- notification campaign control
- React admin sayfaları
- Express route örnekleri
- Prisma model ekleri
- config sözleşmeleri

## Amaç
Deploy sonrası şu işleri panelden yapabilmek:
- tap reward / crit / combo ayarlamak
- görevleri açıp kapatmak
- canlı event başlatmak
- limited offer çalıştırmak
- retention ve revenue metriğini görmek
- notification campaign tetiklemek
- economy dengesini bozmadan canlı test yapmak
