# ADN Production Screen Pack

Bu paket, Telegram Mini App için production'a yakın React ekran iskeletleri içerir.

## Ekranlar
- Home
- Shop
- Missions
- Referral
- Leaderboard
- Chest
- Prestige
- Clan
- Settings

## Ortaklar
- Layout
- Bottom nav
- Stat card
- Section card
- Empty state
- Toast placeholder

## Not
- Tailwind sınıfları kullanıldı
- API çağrıları için placeholder veri eklendi
- `ui/` ve `pages/` altındaki dosyalar doğrudan projeye taşınabilir
