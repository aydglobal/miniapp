# ADN Connected Screen Pack

Bu paket production screen pack'in bir üst katmanıdır.
Amaç: ekranları gerçek API çağrılarına bağlamak.

## İçerik
- api client
- auth/initData header helper
- query hooks
- mutation hooks
- connected pages
- toast ve loading state örnekleri
- types
- route map

## Beklenen backend endpointleri
- GET /profile
- POST /game/tap
- GET /shop
- POST /shop/purchase
- GET /missions
- POST /missions/:id/claim
- GET /referral
- GET /leaderboard/global
- GET /chests
- POST /chests/:id/open
- GET /prestige/summary
- POST /prestige/activate
- GET /clan
- GET /settings
- POST /settings
