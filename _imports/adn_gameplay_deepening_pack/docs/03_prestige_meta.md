# 3. Prestige + Meta Progression

## Hedef
Oyuncu oyunu bitirmiş hissetmesin; reset sonrası daha güçlü başlasın.

## Prestige akışı
- Oyuncu belirli bir güç eşiğine gelir
- Reset yapar
- Normal kart seviyeleri sıfırlanır
- Meta currency kazanır: `Nebula Core`
- Nebula Core ile kalıcı skill tree açılır

## Reset şartı
- Minimum level: 20
- Minimum passive income: 5,000 ADN/hour
- Minimum total earned: 250,000 ADN

## Meta ağaç örnekleri
- Tap Mastery: +%2 permanent tap reward / level
- Crit Lab: +%0.4 crit chance / level
- Energy Reactor: +%3 max energy / level
- Offline Vault: +%5 offline cap / level
- Clan Honor: +%4 clan contribution / level

## Prestige bonus formülü
- `prestigePower = floor(sqrt(totalLifetimeEarned / 100000))`
- Verilen Nebula Core: `3 + prestigePower`
