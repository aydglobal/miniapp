# 5. Event + FOMO Sistemi

## FOMO için gerekenler
- Sınırlı süre
- Görünür sayaç
- Kaçırma hissi
- Tekrarsız ödül

## Hazır event tipleri
- x2 Tap Hour
- Referral Rush Weekend
- Lucky Chest Night
- Prestige Week
- Clan War Sprint

## Event bileşenleri
- Başlangıç ve bitiş zamanı
- Katılım şartı
- Özel çarpan
- Event-only reward
- Event banner + popup + notification

## Limited offer örnekleri
- 10 dakika geçerli starter bundle
- Geri dönen kullanıcıya 1 saat özel offer
- Haftalık ilk satın almaya bonus shard
