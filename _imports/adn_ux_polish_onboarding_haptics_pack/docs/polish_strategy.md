# Polish Strategy

Amaç: oyun hissini "çalışan panel" seviyesinden "premium mini app" seviyesine çıkarmak.

## Ana kalite sütunları
- Görsel hiyerarşi
- Anlık feedback
- Tutarlı motion dili
- Az ama net metin
- Hızlı onboarding
- Ödül anlarında yüksek yoğunluk
- Boş anlarda düşük gürültü

## UI yoğunluk kuralı
- Ana tap alanı: yüksek kontrast, yüksek enerji
- Shop / görev / leaderboard: orta yoğunluk
- Admin / analytics: sade yoğunluk

## Motion dili
- Tap: 120ms - 180ms hızlı scale
- Reward toast: 220ms giriş, 180ms çıkış
- Modal: 240ms fade + rise
- Chest reveal: 650ms staged motion
- Prestige: 900ms çok aşamalı dramatik motion

## Duygusal pik noktaları
- Crit
- Chest drop
- Jackpot
- Level up
- New unlock
- Prestige
- Clan victory
- Event start
