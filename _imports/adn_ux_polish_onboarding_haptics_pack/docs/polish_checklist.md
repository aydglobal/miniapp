# Screen-by-Screen Polish Checklist

## Home
- [ ] Tap orb ekranın odak noktası mı?
- [ ] Balance card okunaklı mı?
- [ ] Event banner var mı?
- [ ] Floating reward text çakışmıyor mu?
- [ ] Combo göstergesi görünür ama rahatsız etmiyor mu?

## Shop
- [ ] Locked kartlar anlaşılır mı?
- [ ] Upgrade etkisi açık mı?
- [ ] Bir sonraki income artışı net mi?
- [ ] Yetersiz bakiye hali düzgün mü?

## Missions
- [ ] 3 ana görev görünür mü?
- [ ] İlerleme çubuğu net mi?
- [ ] Reward claim butonu ayrışıyor mu?

## Referral
- [ ] Kod kopyalama kolay mı?
- [ ] "aktif referral" mantığı açıklanıyor mu?
- [ ] Davet adımı çok uzun değil mi?

## Leaderboard
- [ ] Kendi sıran sabit görünüyor mu?
- [ ] Top 10 ve yakın rakipler ayrı mı?
- [ ] Haftalık / global geçiş kolay mı?

## Chest
- [ ] Açılmadan önce beklenti hissi var mı?
- [ ] Nadirlik net ayırt ediliyor mu?
- [ ] Continue sonrası yönlendirme mantıklı mı?

## Prestige
- [ ] Ne kaybedip ne kazandığı açık mı?
- [ ] Nebula Core net mi?
- [ ] Yanlışlıkla prestige yapmak zor mu?
