# Haptic / Sound Matrix

## Haptic yoğunluk kuralları
- Tap: light
- Combo threshold: light
- Crit: medium
- Chest drop: medium
- Legendary/Mythic reveal: heavy
- Jackpot: heavy
- Error / blocked action: warning pattern
- Level up: success medium
- Prestige: staged medium -> heavy

## Sound yoğunluk kuralları
- Tap: çok kısa, yormayan
- Crit: parlak accent
- Chest reveal: 3 aşamalı yükselen efekt
- Jackpot: imza sesi
- Error: kısa muted
- Success: net ama kısa

## Ses katmanları
- SFX
- UI click
- Reward accents
- Ambient loop (opsiyonel, default kapalı)

## Mobil yorgunluk kuralı
- Tap sesi üst üste spam olmamalı
- 70ms içinde aynı sfx tekrar çalma
- Sessize alma ayarı zorunlu
