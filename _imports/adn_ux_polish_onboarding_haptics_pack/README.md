# ADN UX Polish + Onboarding + Haptic/Sound Integration Pack

Bu paket, ADN tap-to-earn oyununun ürün kalitesini yükseltmek için hazırlanmıştır.

## Kapsam
1. UI/UX polish sistemi
2. Onboarding + tutorial flow
3. Haptic/sound integration
4. Feedback katmanı
5. State ve event bağlama örnekleri
6. Motion ve UI kural setleri
7. Empty state / error state / reward state tasarımları

## Entegrasyon sırası
1. `design-system/` altındaki token ve component kurallarını uygula
2. `frontend/onboarding/` akışını ilk girişe bağla
3. `frontend/feedback/` katmanını tap, chest, level-up ve reward ekranlarına bağla
4. `frontend/audio/` ve `frontend/haptics/` yardımcılarını event bus ile bağla
5. `docs/polish_checklist.md` üzerinden ekran ekran son test yap
