# Component Rules

## Card
- Header + value + subtext düzeni
- En az 20px iç boşluk
- Arka plan iki katmanlı: dark surface + light border

## CTA Button
- Primary: parlak gradient
- Secondary: surfaceAlt + ince border
- Disabled: %45 opacity + glow yok

## Progress Bar
- Arkada koyu ray
- Önde gradient fill
- Yanında net rakam

## Modal
- Üstte icon / state
- Ortada tek güçlü mesaj
- Altta 1 ana CTA + opsiyonel 1 ikinci CTA

## Empty State
- Mini icon
- 1 satır güçlü başlık
- 1 satır kısa açıklama
- 1 net CTA
